So verwenden Sie Frame.io


Auf den Überprüfungsseiten können Sie Feedback zu Videos, Standbildern und PDFs abgeben, ohne ein Frame.io-Konto erstellen zu müssen.
 

Vollständiges Tutorial hier:

Anzeigen von Dateien auf einer Übersichtsseite

Wenn der Link mit einer Passphrase geschützt wurde, werden Sie aufgefordert, diese einzugeben, bevor Sie die Dateien anzeigen können. 

Klicken Sie auf ein beliebiges Asset, um es zu öffnen, spielen Sie den Clip ab, indem Sie die Leertaste drücken oder auf die Schaltfläche "Play" klicken.

Wenn mehrere Dateien mit Ihnen geteilt wurden, können Sie einfach zwischen ihnen hin- und herschalten, indem Sie auf die Pfeile < > klicken. 

Kommentieren

Vorhandene Kommentare (falls vorhanden) werden standardmäßig zusammengeklappt, um eine bessere Ansicht zu ermöglichen. Um die Kommentare zu erweitern, klicken Sie auf das Nachrichtensymbol in der rechten Ecke.

Beim Eingeben eines Kommentars wird das Video eine Pause machen, damit Sie sich Gedanken machen und Ihren Kommentar mit einem Zeitstempel versehen können.
 
 

Bereichsbasierte Kommentare:

Sobald Sie mit der Eingabe Ihres Kommentars beginnen, erscheint ein gelber Selektor in Ihrer Spielerleiste. Ziehen Sie den Griff, um einen Bereich auszuwählen, und klicken Sie auf Enter. Sobald der Kommentar eingereicht wurde, wird der Bereich mit einer grünen Linie auf Ihrer Zeitachse angezeigt.

💡 Kurzanleitung Pro-Tipp: Halten Sie die Shift-Taste gedrückt und verwenden Sie Ihre Links/Rechts-Pfeile für präzises Schrubben.

Drücken Sie die ESC-Taste, um den Bereich zu entfernen, wenn Sie lieber einen Zeitstempel-Kommentar hinterlassen möchten.

Durch Auswahl eines Kommentars wird der Bereich hervorgehoben. Drücken Sie die Leertaste, um den Bereich abzuspielen, oder schalten Sie die Schleife ein, um ihn in einer Schleife abzuspielen.

Drücken Sie die ESC-Taste, um den Wiedergabebereich/die Schleife zu löschen.

Um Ihren Kommentar einzureichen, klicken Sie auf Senden oder drücken Sie die Eingabetaste. Wenn dies Ihr erster Kommentar ist und Sie nicht aktiv bei Frame.io angemeldet sind, werden Sie aufgefordert, Ihren Namen und Ihre E-Mail-Adresse einzugeben. Damit andere Benutzer wissen, wer den Kommentar abgegeben hat, können Sie Ihren Anzeigenamen jederzeit bearbeiten, indem Sie auf das Benutzersymbol oben rechts > Name bearbeiten klicken.

Kommentare können bei Bedarf auch ohne Zeitstempel gesendet werden, indem Sie auf das Häkchen links klicken.

Wenn Sie Ihre Idee nach dem Eingeben des Kommentars weiter vermitteln möchten, wählen Sie das Pinselwerkzeug, um direkt auf dem Video mit verschiedenen Formen und Farben zu zeichnen. Die Zeichnungen haben keinen Einfluss auf die ursprünglich hochgeladene Datei.


Wenn Sie auf einen Kommentar klicken, gelangen Sie genau zu dem Rahmen, in dem er hinterlassen wurde. Die violetten Punkte entlang Ihrer Zeitachse zeigen Kommentare an, fahren Sie mit der Maus über einen, um den Kommentar anzuzeigen, und klicken Sie auf diesen, um zu diesem Rahmen zu gelangen. 🎉 

Markieren Sie einen Kommentar als vollständig, indem Sie auf den Kreis klicken. Wählen Sie die Option zum Ausblenden/Anzeigen abgeschlossener Kommentare.

Sortieren Sie Ihre Kommentare nach Ältestem, Neuestem, Timecode oder nach Kommentator.

Beschriftungen

Wenn Sie den Status für die Datei festlegen müssen, wird die Option oben rechts angezeigt. Wählen Sie zwischen Genehmigt, In Bearbeitung oder Bedarfsüberprüfung. Sobald Sie eines dieser Labels ausgewählt haben, wird eine Benachrichtigung an alle Benutzer des Projekts gesendet.

Herunterladen

Wenn Sie Zugang zum Herunterladen der Dateien erhalten haben, sehen Sie die Schaltfläche zum Herunterladen in der rechten oberen Ecke. Wenn Sie auf Download klicken, wird die Originaldatei heruntergeladen, nicht ein Proxy. 🌟
